# PUC 2024/2 - Testes de Aplicações Mobile: Android

Criação do nosso primeiro App Multiplataforma usando o React Native.

## Comandos Executados

```bash
npm install -n
npm install -G npm
npx create-expo-app@latest
cd .\puc-2024-2-tam-catherine-react-native
npm run web
```
